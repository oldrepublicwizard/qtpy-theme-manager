# qtpy-theme-manager

Apply QSS/palette/style themes to a QApplication; discover QRC and VS Code JSON themes; host app owns persistence (see THEME_MIGRATION.md).

## Install

```bash
pip install -e .
# or from GitHub:
pip install git+https://github.com/oldrepublicwizard/qtpy-theme-manager
```

## Origin

Extracted from the [PyKotor](https://github.com/OpenKotOR/PyKotor) monorepo `utility` / related packages.
KotOR-specific couplings were removed or made optional for standalone use.
